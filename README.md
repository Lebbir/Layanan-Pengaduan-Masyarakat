# LaporDesa - Platform Pengaduan Masyarakat




LaporDesa adalah platform digital untuk menyalurkan keluhan dan aspirasi warga desa dengan sistem yang mudah, cepat, dan transparan. Platform ini memungkinkan masyarakat untuk menyampaikan laporan dan memantau perkembangan laporan mereka secara real-time.
